justify-content - This CSS property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container.

align-items - This  property sets the align-self value on all direct children as a group. In Flexbox, it controls the alignment of items on the Cross Axis. In Grid Layout, it controls the alignment of items on the Block Axis within their grid area.

gap - The gap CSS shorthand property sets the gaps (gutters) between rows and columns.

flex-direction - The flex-direction CSS property sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed).

flex-wrap - The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.
